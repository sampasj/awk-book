# bmi
BMI を計算する。  
体重をキログラム、身長をメートルで入力。 p.29  

```
$ ./bmi
kilograms meters
100 1.67
35.9
71 1.67
25.5

```

# mkdata.sh

本では著者の１人[^1]の歩数の実データとのことだが、そんなデータは持ち合わせていないのでダミーデータを作成する。  

```
$ ./mkdata.sh | head
2022/1/1 19480
2022/1/2 12878
2022/1/3 7127
2022/1/4 12578
2022/1/5 15802
2022/1/6 16738
2022/1/7 9727
2022/1/8 7643
2022/1/9 16577
2022/1/10 3017
$ ./mkdata.sh > steps.dat

```

[^1]: カーニハンに決まっている
